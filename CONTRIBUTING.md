# Contributing

Thanks for helping improve Spookyfin.

Useful contributions:

- Jellyfin client compatibility notes.
- CSS fixes for newer Jellyfin web UI releases.
- Screenshots on different screen sizes.
- Additional library art in the same style.
- Accessibility fixes for contrast, focus states, and motion.

Please keep the base theme CSS-first. JavaScript changes should stay limited to
`spookyfin-helper.js`, which handles browser-only enhancements such as the color
switcher and image rehydration.
